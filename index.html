Summary

The purpose of this project was to analyze how Price changes based on usage and year of vehicles. To accomplish this analysis, we first pulled data from the craigslist to assemble a dataset on over 500,000 vehicles.

After assembling the dataset, we used Pandas Profiling Libary to profile the data and Pandas Matplotlib to plot various aspects of the Odometers and Years vs. Price. Factors we looked at included: condition, fuel, and cylinders. This site provides the source data and visualizations created as part of the analysis using Tableau, as well as explanations and descriptions of any trends and correlations witnessed.
